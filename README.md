<h2 align="center">like</h2>
一个跨平台音视频直播应用，目前支持 Windows x64 / macOS 操作系统，本项目依赖于腾讯实时音视频 SDK 和 WebRTC 以及 SRS 流媒体服务等技术，实现语音视频通话和在线互动直播。
