# like
WebRTC音视频直播应用

## 项目介绍
- 前端技术栈：Vue + Electron + WebRTC + Socket.io-client
- 后端技术栈：[SRS](https://github.com/ossrs/srs) + [likesignal](https://github.com/themages/likesignal)

本仓库后期会进行重要版本的重构，目前以实现核心的需求功能为主要目标。
